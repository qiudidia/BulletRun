### 踩坑经验（补充）

- GDScript 4 中，同一函数内 `if` 分支内外各声明一次 `var x` 会产生 "declared below in the parent block" 警告，必须改名或统一声明
- `CharacterBody2D` 在 `take_damage()` → `_die()` 链路中如果 `_die()` 调用 `reload_current_scene()`，而 `take_damage` 是从 `body_entered` 物理信号触发，就会报 "Removing a CollisionObject during physics callback" → 必须用 `call_deferred` 包装
- `Polygon2D` 可以通过代码设置 `polygon = PackedVector2Array([...])` 来画圆形，比生成 PNG 贴图更轻量，适合程序化角色外观
- `ConfigFile.get_value(section, key, null)` 在 Godot 4 中如果 `section` 不存在且 `default` 为 `null`，会报 "no default was given" → 确保所有 key 都在 `DEFAULTS` 中有默认值，或者在 `get_value` 中传递非 null 默认值
