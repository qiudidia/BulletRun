# 2026-07-03 工作日志

## 用户建议实现

### 1. 新僵尸品种（已完成）
添加了3个新品种僵尸：
- **速度型(Runner)** - 浅绿色，30血，150速度，10伤害，白色眼睛
- **坦克型(Tank)** - 深绿色，200血，60速度，30伤害，红色眼睛
- **爆炸型(Bomber)** - 黄色，80血，90速度，20伤害，橙色眼睛，死亡时爆炸（范围伤害40，半径80）

**修改的文件：**
- `scripts/ZombieAI.gd` - 添加 `zombie_type` 变量，修改 `_ready()` 根据类型设置属性，修改 `_die()` 添加爆炸逻辑，修改 `_create_eyes()` 根据类型设置眼睛颜色
- `scenes/game/zombie_mode/ZombieGame.gd` - 修改 `_spawn_wave_enemies()` 根据波次生成不同品种，添加 `_get_zombie_name()` 函数
- `scenes/game/multiplayer/ZombieCoopGame.gd` - 修改 `_spawn_wave_enemies()` 和 `_spawn_wave_from_data()` 支持新类型，添加 `_get_zombie_name()` 函数
- `Autoload/GameSettings.gd` - 添加新品种僵尸的翻译键

**生成规则：**
- 波次1-2：主要普通僵尸
- 波次3+：可能混入速度型（30%概率）
- 波次5+：可能混入坦克型
- 波次7+：可能混入爆炸型

### 2. 宝箱系统（单机版已完成）
在地图上随机刷新宝箱，玩家经过自动拾取获得金钱。

**修改的文件：**
- `scripts/TreasureChest.gd` - 创建宝箱脚本（金色视觉效果，光效脉动动画，拾取后20-40秒刷新）
- `scenes/game/zombie_mode/ZombieGame.gd` - 添加宝箱系统（5个宝箱同时存在于地图，每15秒检查一次是否需要生成新的）

**功能：**
- 宝箱在随机位置生成（避开玩家初始位置）
- 玩家经过自动拾取，获得50-150随机金钱
- 拾取后显示金色通知 "+$XX"
- 宝箱自动刷新（20-40秒后重新出现）
