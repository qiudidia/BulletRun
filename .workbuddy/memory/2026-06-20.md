# 2026-06-20 工作记录

## Bullet Run - 全面 Bug 审查与修复

Lint 全绿（0 错误），人工逻辑审查发现并修复 6 个 Bug：

### Bug 修复清单

- **E047**: `ZombieGame.gd` `_init_shop_ui` 中存在两个未使用变量 `title_idx` / `money_idx` → 删除
- **E048**: `ZombieGame.gd` `_game_over()` 只 emit 信号但没有实际切场景，玩家死亡后啥也不发生 → 加 `await 1秒` + `change_scene_to_file("res://scenes/menu/main_menu.tscn")`，同时重置金钱
- **E049**: `ZombieGame.gd` `_ready()` 从 `GameSettings` 读取 `money`，导致金钱跨局累积 → 改为每局重置 `money = 0`，同时写入 `GameSettings.set_value("game", "money", 0)`
- **E050**: `BotGame.gd` `_spawn_bot` 中备用临时 bot 颜色是蓝色（`Color.BLUE`）→ 改为红色（`Color(0.9, 0.2, 0.2, 1)`）
- **E051**: `BotGame.gd` `_get_random_spawn_position` 使用 `while` 死循环，极端情况（地图很小）可能卡死 → 改为最多尝试 20 次的 `for` 循环
- **E052**: `ZombieAI.gd` / `BotAI.gd` `_die()` 直接调用 `queue_free()`，在子弹 `body_entered` 链路中会产生 "Removing CollisionObject during physics callback" 警告 → 改为 `call_deferred("queue_free")`

### 重要路径确认

- 主菜单场景路径：`res://scenes/menu/main_menu.tscn`（不是 `MainMenu.tscn`）
