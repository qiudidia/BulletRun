# Bullet Run 🔫

一个基于 Godot 4.6 的俯视角射击游戏，支持单机模式和联机对战。

## ✨ 特性

- **多种游戏模式**
  - 🧟 僵尸模式：合作生存，抵御越来越强的僵尸潮
  - 🤖 Bot模式：与AI对手对战，提升技能
  - ⚔️ 单挑模式：1v1对决
  - 🥊 乱斗模式：3人自由对战
  - 🧟‍♂️ 僵尸合作：多人合作抵御僵尸

- **丰富的武器系统**
  - 手枪、步枪、狙击枪、刀、机枪
  - 每种武器都有独特的手感和战术用途

- **装备配置与特长**
  - 5套装备配置
  - 3类特长：移动、防御、战斗

- **XP与等级系统**
  - 16级军衔系统
  - 击杀获得XP，升级解锁新军衔

- **联机功能**
  - 基于ENet的局域网联机
  - UDP广播自动发现房间
  - 中途加入游戏

## 🎮 操作说明

| 按键 | 功能 |
|------|------|
| WASD | 移动 |
| 鼠标左键 | 射击 |
| R | 换弹 |
| 1-5 | 切换武器 |
| E | 交互/打开商店 |
| G | 投掷手雷 |
| M | 切换小地图 |
| ESC | 暂停 |

## 🚀 下载与安装

### 方法一：使用启动器（推荐）
1. 下载 `BulletRun-Launcher.zip`
2. 解压后运行 `Launcher.exe`
3. 启动器会自动检查更新并下载最新版本

### 方法二：手动下载
1. 前往 [Releases](https://github.com/你的用户名/BulletRun/releases) 页面
2. 下载最新版本的 `BulletRun-vX.X.X-win64.zip`
3. 解压后运行 `BulletRun.exe`

## 🔧 从源码构建

1. 安装 [Godot 4.6](https://godotengine.org/download)
2. 克隆本仓库：
   ```bash
   git clone https://github.com/你的用户名/BulletRun.git
   ```
3. 用Godot打开项目文件夹
4. 点击右上角 **导出** 按钮，选择平台并导出

## 📋 系统要求

- **操作系统**：Windows 10/11, macOS, Linux
- **处理器**：2.0 GHz 双核
- **内存**：4 GB RAM
- **显卡**：支持OpenGL 3.3的显卡
- **存储空间**：500 MB 可用空间
